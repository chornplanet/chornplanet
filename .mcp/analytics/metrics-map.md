# Analytics Metrics Map

This map lists approved metric categories for future on-demand analytics access.

## Google Search Console

Aggregate metrics:

- clicks
- impressions
- click-through rate
- average position
- top queries
- top pages
- country and device breakdowns when useful
- indexed pages and coverage issues when available

Planning questions:

- Which pages have high impressions and weak CTR?
- Which search queries deserve new content?
- Which pages rank but need better metadata?
- Which topic hubs are gaining visibility?

## Google Analytics

Aggregate metrics:

- users
- sessions
- page views
- engagement rate
- traffic source
- route or content category performance
- events when already instrumented
- outbound product-link clicks when approved and available

Planning questions:

- Which pages keep users engaged?
- Which categories drive repeat visits?
- Which traffic sources are growing?
- Which conversion or outbound click events need instrumentation?

## Vercel / Speed Insights

Aggregate signals:

- route performance issues
- page speed regressions
- deployment impact
- user-experience performance patterns

Planning questions:

- Which pages need performance work?
- Did a release affect speed or engagement?
- Which route family should be optimized first?

## Future Approved Signals

Potential later sources:

- TikTok click/performance signals
- Shopee/Lazada/Amazon click signals
- owned commerce conversion summaries
- Smart Food web behavior summaries

These require explicit approval before integration.
