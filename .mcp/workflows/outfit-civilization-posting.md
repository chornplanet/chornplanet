# Outfit and Civilization Posting Workflow

Use this workflow for lady/gentleman outfit, civilization clothing, and ChornPlanet visual commerce posts.

## Goal

Use DNA to create premium outfit/civilization content that can grow traffic and support TikTok-first commerce.

## Required authority

Before creating outfit/civilization generation plans, read DNA authority:

```text
.dna/.chatgpt/Agents.md
.dna/.chatgpt/System/World/Zones/Zone List.md
```

## Command pattern

```text
StoryGenProduct: <attachments>
AutoScene: [...]
Mode: [Word/.../...]
SkipVideo: True or False
PostKeywords: []
```

## Zone resolution

```text
Zone List -> Zone File -> exactly one Sub-Zone
```

Rules:

- Zone List is only the entry point.
- Select one zone file.
- Select exactly one Sub-Zone block.
- Lock the Sub-Zone across all six scenes.
- Preserve environment identity, object system, lighting behavior, and mood direction.
- Vary pose, camera, emotion, foreground object, background emphasis, and interaction.

## Six-scene output direction

- Hook: strongest outfit/product + environment hero view
- Scene1: arrival / discovery
- Scene2: lifestyle interaction
- Scene3: seated / use-case / deeper activity
- Scene4: transition / destination movement
- Closing: final hero / emotional brand ending

## Commerce connection

Every outfit/civilization content plan should optionally support:

- product identity
- product page draft
- TikTok-first post draft
- marketplace link placeholder
- analytics tracking idea

## Approval rule

Drafting content is allowed.

External publishing or marketplace product linking requires approval.
