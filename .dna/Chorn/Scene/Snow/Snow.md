# Identity

Luxury **living corner viewpoint** overlooking a majestic **Snow Landscape**.

The scene represents a **realistic human living corner integrated with an alpine winter environment**, such as a snow
lodge terrace, glass viewing platform, alpine deck, or winter garden corner.

Foreground living corner → snow valley or pine forest → frozen landscape → snow mountain horizon.

The snow biome represents **peaceful alpine winter nature**, featuring soft snow layers, frozen rivers, alpine pine
forests, and majestic glacier mountains.

The purpose of this scene system is to create **cinematic luxury winter campaign visuals where human life peacefully
coexists with pristine alpine snow landscapes**.

---

# Invocation Command

```
Snow: {scene label}
```

Examples

```
Snow: Alpine Snow Valley Terrace

```

The invocation command references the **scene label** rather than a number, enabling **AI Scene Engine** to interpret
the winter environment correctly.
