# 3. View Mode System

Catappa scenes support two cinematic view modes.

---

## Default View (Panoramic River View)

Activated when user simply calls:

```
Mountain Catappa x
```

This produces compositions like **Reference#2 of Catappa 2**

Characteristics

• Long river perspective
• Multiple catappa trees visible
• Long flower fields along river bank
• Wooden huts clearly visible across river
• Full circular platform visible
• Large environmental scale

Priority structure

```
Flower meadow → Catappa tree corridor → River → Wooden huts → Mountains
```

This mode emphasizes **grand landscape composition**

---

## Half Circular View

Activated when user specifies

```
Mountain Catappa x - half circular
```

Produces images similar to **Reference#1 of Catappa 2**

Characteristics

• Platform appears half circular
• Camera closer to tree
• Giant catappa tree dominates frame
• Lantern clearly visible
• Flower meadow still present
• River visible but shorter perspective

Priority structure

```
Tree canopy → Lantern → Half circular platform → Flower meadow → River
```

This mode emphasizes **tree architecture focus**

# 4. Scene Stability Engine (NEW in Catappa-v5)

Catappa-v5 introduces **Scene Stability Engine**.

This system prevents AI from drifting away from the core scene structure.

The environment is locked using **five structural anchors**.

---

## Structural Anchor System

Every Catappa scene must include the following anchors.

### Anchor 1 — Tree Corridor

Giant **Terminalia catappa trees** aligned along the riverbank.

Requirements

• Symmetrical spacing
• Trees extend toward horizon
• Large umbrella canopy

---

### Anchor 2 — Circular Platform

Each tree stands on a **circular teak platform**

Platform components

• Teak wood deck
• Circular geometry
• Access stairs
• Waist-height glass guardrail

Capacity

1–2 persons

---

### Anchor 3 — Flower Bank

Flower field must occupy the **entire front river bank**

Requirements

• Continuous flower bank
• No empty soil gaps
• Large cinematic coverage

Density rule

70–90% ground coverage.

---

### Anchor 4 — River Corridor

Behind the trees lies a **crystal-clear mountain stream**

Requirements

• Visible river depth
• Flowing water surface
• River perspective stretching into distance

---

### Anchor 5 — Hut Alignment

Across the river must appear

**rows of wooden mountain huts**

Requirements

• huts aligned along river bank
• multiple huts visible
• architectural consistency

---

# 5. Environmental Layout (Locked)

The environmental order must remain consistent.

```
Flower meadow
→ Catappa tree corridor
→ Crystal mountain stream
→ Wooden huts
→ Mountain range
```

# 6. Lantern System

Each tree contains a hanging

**White Lanna lantern**

Specifications

• pure white color
• geometric Lanna design
• suspended rope
• tassel bottom

Lanterns must hang under the canopy.

---

# 7. Flower Rule

Each scene uses **one flower species only**

Do not mix species.

---

# 8. Flower Density Rule

Flower fields must be **visually dominant**

Coverage target

70–90% of visible ground.

Purpose

Create large cinematic flower landscapes.

---

# 9. River Perspective Rule

Default Catappa scenes must show **long river perspective**

Requirements

• river visible extending into distance
• riverbank flower fields
• tree corridor perspective

This rule applies to **Panoramic View**

---

# 10. Scene Scale Rule

Environment must remain **large scale**

Requirements

• giant trees
• wide river valley
• long tree corridor
• large flower fields

Do not generate small garden scenes.
