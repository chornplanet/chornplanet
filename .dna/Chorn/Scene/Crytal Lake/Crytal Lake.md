# Identity

Luxury **living corner viewpoint** overlooking a majestic **Crystal Lake Landscape**.

The scene represents a **realistic human living corner integrated with pristine alpine lake environments**, such as a lakeside terrace, glass viewing deck, wooden pier lounge, or crystal lake garden platform.

Foreground living corner → crystal lake shoreline → clear water reflections → alpine mountain horizon.

The Crystal Lake biome represents **pure alpine freshwater nature**, featuring crystal-clear lakes, reflective water surfaces, alpine forests, and calm mountain landscapes.

The purpose of this scene system is to create **cinematic luxury lake retreat visuals where human life peacefully coexists with pristine crystal lake environments**.

---

# Invocation Command

```
CrystalLake: {scene label}
```

Examples

```
CrystalLake: Crystal Lake Terrace
CrystalLake: Alpine Reflection Deck
CrystalLake: Lakeside Garden Viewing Corner
```

The invocation command references the **scene label** rather than numbers, allowing **AI Scene Engine** to interpret lake environments correctly.
