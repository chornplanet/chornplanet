# Identity

Luxury **living corner viewpoint** overlooking a majestic **Sky Island Landscape**.

The scene represents a **realistic human living corner integrated with floating sky island environments**, such as a sky
terrace, floating garden deck, glass sky balcony, or aerial lodge platform.

Foreground living corner → floating island gardens → cloud ocean → distant sky island mountains.

The Sky Island biome represents **peaceful high-altitude nature**, featuring floating landforms, sky forests, cloud
seas, and luminous atmospheric skies.

The purpose of this scene system is to create **cinematic luxury sky campaign visuals where human life peacefully
coexists with floating sky island landscapes**.

---

# Invocation Command

```
Sky: {scene label}
```

Examples

```
Sky: Floating Island Terrace
Sky: Cloud Ocean Panorama Deck

```

The invocation command references the **scene label** rather than numbers, enabling **AI Scene Engine** to interpret sky
environments correctly.
