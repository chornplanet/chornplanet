# Identity

Luxury **living corner viewpoint** overlooking a majestic **Flower Desert Landscape**.

The scene represents a **realistic human living corner integrated with desert environments where seasonal flowers bloom across sand valleys**, such as a desert terrace, stone viewing platform, oasis garden deck, or dune lodge corner.

Foreground living corner → desert flower valley → sand dunes → desert horizon.

The Flower Desert biome represents **rare desert bloom ecosystems**, where vast sand landscapes are combined with colorful flower fields that appear seasonally across desert terrain.

The purpose of this scene system is to create **cinematic luxury desert bloom campaign visuals where human life peacefully coexists with dramatic desert flower landscapes**.

---

# Invocation Command

```
FlowerDesert: {scene label}
```

Examples

```
FlowerDesert: Desert Bloom Terrace
FlowerDesert: Sand Dune Flower Panorama
FlowerDesert: Oasis Garden Viewing Corner
```

The invocation command references the **scene label** rather than numbers, enabling **AI Scene Engine** to interpret desert bloom environments correctly.
