# 2. DNA Structure (5 Main Layers)

---

## Layer 1: Nature as Spiritual Foundation (Non-Negotiable)

### Principle

ธรรมชาติไม่ใช่ Background
ธรรมชาติคือ “พลังหลักของภาพ”

### Required Elements

* Mountain depth
* River / Water presence
* Mist / Atmospheric diffusion
* Layered foreground-midground-background

### Rule

ภาพต้องมี “Air Perspective”
ถ้า Background แบน = DNA ไม่สมบูรณ์

### Emotional Output

* Calm
* Sacred
* Timeless

---

## Layer 2: Textile as Luxury Language

### Principle

ผ้า = ภาษาของความหรู

### Requirements

* ลายต้องอ่านได้
* Texture ต้องเห็น
* Reflection ของเครื่องเงินต้องมีน้ำหนักจริง
* Fabric Flow ต้องเป็นธรรมชาติ

### Upgrade Rule

Detail = Status Signal

ถ้า Textile ดูเรียบเกินไป
Luxury จะลดระดับทันที

---

## Layer 3: Symbolic Narrative System

### Principle

Luxury ที่แท้จริงต้องมีความหมาย

### ทุกภาพต้องมีอย่างน้อย 1 Symbol

| Symbol   | Meaning             |
|----------|---------------------|
| Water    | Purification / Flow |
| Mist     | Mystery / Memory    |
| Mountain | Stability / Legacy  |
| Lantern  | Ritual / Warmth     |
| Bird     | Spirit / Freedom    |
| Sand     | Passage of Time     |
| Jewelry  | Cultural Wealth     |

### Rule

Symbol ต้อง “เป็นธรรมชาติ” ไม่ใช่ของตกแต่ง

---

## Layer 4: Emotional Geometry

### Principle

ตำแหน่งร่างกาย = ความหมาย

### Single Portrait Modes

* Standing Still → Heritage
* Looking Side → Grace
* Slight Smile → Soft Power
* Chin Up → Authority

### Couple Modes

* Walking → Future
* Standing Close → Bond
* Shared Direction Gaze → Legacy
* Slight Physical Distance → Respectful Love

---

## Layer 5: Future + Heritage Contrast

### Principle

ความล้ำค่าเกิดจากการตัดกันของยุคสมัย

### Implementation Patterns

* Modern Male + Traditional Female
* Clean Minimal Styling + Rich Textile
* Barefoot + Royal Fabric
* Cinematic Light + Cultural Costume

### Rule

ถ้าไม่มี Contrast
ภาพจะกลายเป็น “ย้อนยุคธรรมดา”

# 3. Visual Construction Rules

---

## 3.1 Depth Construction

ภาพต้องมี 3 ระดับ:

1. Foreground (ดอกไม้ / ทราย / โคมไฟ)
2. Subject (ตัวแบบ)
3. Background (ภูเขา / หมอก / ป่า)

---

## 3.2 Light Architecture

Preferred Lighting:

* Soft morning light
* Warm golden diffusion
* No harsh shadow
* Skin tone natural

Avoid:

* Flat lighting
* Studio flash look
* Overexposed white sky

---

## 3.3 Color Philosophy

Base Palette:

* Earth tone
* Deep red / Maroon
* Muted gold
* Soft lavender
* Forest green

Avoid:

* Neon
* Over-saturated color
* High contrast fashion lighting

---

Luxury ในระบบนี้ไม่ใช่:

* Meaning

### ✅ 5. Emotional Mode ชัดไหม?
