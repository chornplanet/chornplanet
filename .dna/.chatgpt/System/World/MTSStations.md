# D. Station Layer Role

Stations represent the **transition layer between infrastructure and environment scenes**.

Stations function as:

```
rail access points
zone entry portals
tourism arrival gateways
urban commuting hubs
```

Every zone must contain at least:

```
1 primary station
```

Some zones may contain multiple stations for tourism environments.

---

# E. Station Architecture Rule

All stations must visually include:

```
platform rails
train boarding platforms
rail alignment
station structures
passenger circulation areas
```

Stations must visually integrate with the surrounding environment.

Examples:

Mountain

```
wood + stone alpine structures
mist valley platforms
```

Urban

```
glass architecture
city plaza integration
```

Coastal

```
open-air tropical station
ocean horizon views
```

# G. Flower Identity System

Each station includes a **flower identity**.

Purpose:

```
environment storytelling
visual differentiation
biome identity
```

Rule:

```
1 flower species per scene
```

Examples:

```
Mountain Lavender Valley → lavender
Sakura Station → sakura blossoms
River Valley Station → river valley wildflowers
```
