```text
every tourism scene must feel like a real destination inside a living world
```

Therefore, each scene must be more than visual decoration.

```text
place to arrive
```

Tourism scene design must support both

```text
visual storytelling
and
destination logic
```

at the same time.

# E. Global Tourism Place Design Principles

## 1. Living Tourism Civilization Rule

Every tourism zone must be designed as part of a living tourism world.

Shared core components:

```text
tourism
mobility
economy
leisure
landmark identity
destination structure
```

Tourism scenes must imply a functioning place, not an isolated image.

---

## 2. Cinematic Living Corner Rule

Every scene must function as a

```text
cinematic living corner
```

This means the viewer should feel that the place

```text
can be entered
can be photographed
can host story continuation
can support image and video production
can operate as a tourism destination
```

---

## 3. Tourism Realism Rule

Every tourism place must feel like a believable world-class destination.

Required qualities:

```text
destination-ready
visitor-friendly
economically alive
architecturally believable
operationally believable
```

---

## 4. Zone Scalability Rule

Every tourism zone must support future expansion.

Its structure should be scalable toward

```text
new stations
new viewpoints
new landmark districts
new tourism sub-zones
new leisure areas
```

This rule supports long-term world growth without redesigning the foundation.

---

# F. Zone Tourism Structure Standard

## 1. 100-Scene Rule

Every major tourism zone may use the standard:

```text
100 scenes per zone
```

Recommended structure:

```text
10 stations
10 scenes per station
```

Total:

```text
100 scenes
```

This is a **design planning standard**, not a world registry rule.

The actual official station registry remains under the World Layer.

---

## 2. Station Identity Rule

Each station should carry a

```text
clear tourism identity
```

Examples:

```text
Lavender Valley
Coral Bay Marina
Ocean Cliff Horizon
Grand Alpine Terminal
```

A station functions as a

```text
major destination district
```

within a larger tourism zone.

---

## 3. Station Scene Pattern Rule

The recommended 10-scene station pattern is:

```text
01 Station Arrival Scene
02 Main Viewpoint
03 Scenic Walkway
04 Landmark Structure
05 Photo Platform
06 Café / Restaurant Terrace
07 Nature / Garden / Waterfront Corner
08 Activity Zone
09 Leisure Lounge
10 Cinematic Panorama Scene
```

This sequence may adapt to the zone, but should preserve the role structure:

```text
arrival
view
walk
landmark
photo
economy
nature
activity
leisure
panorama
```

---

## 4. MTS Presence Rule

MTS should appear primarily in

```text
station arrival scenes
```

This helps preserve realism:

```text
10 MTS scenes
90 non-train tourism scenes
```

for a typical 100-scene tourism zone.

The exact mobility authority remains in:

```text
MTSNetwork
MTSLines
MTSStations
```

This file only defines tourism design usage.
