# Purpose

This file is the master index for the Chorn Planet ecosystem folder.

It exists to:

* define the ecosystem document set
* reduce document size
* separate strategic sections by priority
* make review and approval easier
* allow phased development one file at a time

# Phase-1 Rule

Phase 1 should focus on:

* ecosystem identity
* architecture
* existing commerce reality
* current TikTok and LINE OA integration
* strategic map direction
* strategic mixed-use direction

Phase 1 should avoid too much detail in:

* full city planning
* detailed real-estate specification
* partner ecosystem design
* advanced analytics architecture
